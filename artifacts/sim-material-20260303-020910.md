# Convolutional Neural Networks
## Convolution Operation
- Kernels slide over local receptive fields.
- Stride controls movement and output size.

## Padding
- Zero-padding preserves spatial dimensions.
- Valid vs same padding changes feature map size.

## Pooling
- Max pooling keeps strongest activation.
- Average pooling smooths responses.

## Feature Maps
- Early layers detect edges and textures.
- Deeper layers capture semantic structures.

## Backpropagation in CNNs
- Gradients flow through convolutions and pooling paths.
- Parameter sharing reduces trainable weights.

## Regularization
- Dropout and data augmentation reduce overfitting.
- Weight decay can improve generalization.
